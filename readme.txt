A quick, easy way to add an Responsive slider with content OR Responsive slider inside wordpress page OR Template. Also mobile tuch slider with content

== Description ==

This plugin add a Responsive slider with content in your website. Also you can add Responsive slider page and mobile tuch slider in to your wordpress website.

== Installation ==

For a manual installation via FTP:

1. Extract the downloaded plugin and upload to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

1. You can also [download](https://downloads.wordpress.org/plugin/wp-dc-slider.zip "Download Dc Responsive content slider") the plugin.
2. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
3. Activate the plugin through the 'Plugins' screen in your WordPress admin area
